## About

Dys-Kop is a production Next.js 15 website built for a mini-excavator services company operating in the Lublin region of Poland. The site is fully server-rendered with App Router, uses Payload CMS v3 for gallery management backed by Neon Postgres and Vercel Blob storage, and is optimized for Polish-language SEO with structured data, sitemaps, and semantic HTML.

## SEO Results

SEO was a top priority for the client. The site achieves a **Lighthouse SEO score of 100** and ranks **#1 on Google** for target keywords like *"uslugi minitraktorem lubelskie"* (mini-tractor services Lublin region).

## Admin Panel (Payload CMS)

Gallery management powered by [Payload CMS v3](https://payloadcms.com/). Image metadata (alt text, display order) stored in Neon Postgres, files in Vercel Blob. Accessible at `/admin`.
